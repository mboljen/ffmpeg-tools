# ffmpeg-splitaudio

Split monolithic audio book into chapter files


## Synopsis

```console
$ ffmpeg-splitaudio [OPTION]... INFILE...
```


## Description

[here]


## Options

+ `-t`, `--to` FORMAT

  Set target format (defaults to format of input file)

+ `-r`, `--readonly`

  Extract chapter information only

+ `-y`, `--yes`

  Overwrite existing files
